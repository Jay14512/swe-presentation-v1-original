## üìå Project Overview

This exercise focused on the early stages of software development: planning, structuring, and communicating an application before writing any code.  
The goal was to design a complete web application idea using PHP and SQL, define the technical requirements, and present the planned architecture.

As part of the assignment, we had to:

- Explain the app concept and its core functionality  
- Describe how users interact with the system  
- Outline the server-side logic, APIs, and database usage  
- Present the planned technical stack  
- Create a detailed **Entity Relationship Model (ERM)**
